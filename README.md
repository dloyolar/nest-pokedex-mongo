# Run in development

1. Clone repository
2. Execute

```
yarn install
```

1. Install Nest CLI in your OS

```
npm i -g @nestjs/cli
```

4. Run Database

```
docker-compose up -d
```

4. Build DB with Seed

```
http://localhost:3000/api/v2/seed
```
